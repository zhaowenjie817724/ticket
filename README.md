# 极速抢

极速抢是一个跨平台合规购票作战台，同一套项目支持 `12306` 和 `大麦` 两种模式切换。

它的目标是把合法范围内的成功率拉满：提前检查资料、倒计时、生成备选策略、打开官方入口，并让你在关键节点本人确认。项目不实现自动锁票、自动提交订单、验证码识别、接口逆向、代理池或多账号并发。

## 运行和发布

```powershell
cd H:\极速抢
npm test
npm start
```

启动后打开：

```text
http://127.0.0.1:5177/
```

手机和电脑连同一个 Wi-Fi 后，终端还会打印类似下面的局域网地址：

```text
http://192.168.x.x:5177/
```

用 Android Chrome 或 iOS Safari 打开这个地址，就可以添加到桌面/主屏幕。也可以直接双击 `index.html` 打开。若要使用 PWA 安装和离线缓存，请用 `npm start` 后通过浏览器访问本地或局域网地址。

正式给别人使用时，不要依赖同一个 Wi-Fi。把项目部署到 Vercel、Netlify、Cloudflare Pages 或 GitHub Pages，然后分享公网链接。详细步骤见 `DEPLOY.md`。

## 平台覆盖

- 电脑：Windows/macOS/Linux 浏览器。
- Android：Chrome/Edge 浏览器访问本地或局域网地址，可添加到桌面。
- iOS：Safari 浏览器访问本地或局域网地址，可添加到主屏幕。

## 模式

### 12306

- 出发地、目的地、出行时间。
- 席别优先级。
- 起售时间和候补截止时间。
- 相邻站、中转换乘、席别降级、前后一天等备选策略。
- 官方入口：12306 官网和官方余票查询。

### 大麦

- 演出名称、城市、演出时间。
- 开售时间。
- 票档优先级。
- 预算、观演人数、加场/二次放票/退票回流等备选策略。
- 官方入口：大麦官网和 App 页面。
