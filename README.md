# 极速抢索票台

极速抢索票台是一个面向客户交付的官方购票需求整理工具，支持 `12306` 和 `大麦` 两种模式。它不做自动抢票，而是把客户票需整理成可复制、可分享、可确认的索票单，并生成官方网页和手机 App/移动入口。

项目不实现自动锁票、自动提交订单、验证码识别、接口逆向、代理池或多账号并发。

## 运行和发布

```powershell
cd H:\极速抢
npm test
npm start
```

启动后打开：

```text
http://127.0.0.1:5177/
```

手机和电脑连同一个 Wi-Fi 后，终端还会打印局域网地址，可以用手机测试官方 App/移动入口。

## 客户使用流程

1. 选择 `12306` 或 `大麦`。
2. 上传客户截图或粘贴截图文字，自动提取核心票需。
3. 补齐客户姓名、联系方式、乘车人/观演人和备选方案。
4. 点击 `生成索票单`。
5. 把右侧索票单复制或分享给客户确认。
6. 客户确认后，点击官方网页或官方 App/移动入口完成本人登录、下单和支付。

## 图片填表

- Vercel 部署时，`/api/extract-ticket` 会读取环境变量调用图片模型，不把 Key 放进前端。
- 静态站点没有 API 时，前端会尝试浏览器 OCR；不可用时仍可粘贴截图文字并自动解析。
- 支持从 12306 截图提取行程、时间、人数、席别、乘车人、候补截止。
- 支持从大麦截图提取项目、城市、演出时间、开售时间、票档、预算、观演人、项目 ID。

可选环境变量：

```text
ANYROUTER_API_KEY
ANYROUTER_BASE_URL
ANYROUTER_MODEL   # 必填，需是该网关实际支持图片输入的模型名
MUYUAN_API_KEY
MUYUAN_BASE_URL
MUYUAN_MODEL
```

## 12306 模式

- 客户信息、出发地、目的地、出行日期、人数。
- 乘车人/证件备注。
- 可接受席别、时间窗口、候补截止和备选方案。
- 可填写出发站/到达站电报码，生成官方余票目标链接。
- 手机端提供 12306 App 深链、Android intent 和官方网页兜底。

## 大麦模式

- 客户信息、演出名称、城市、演出时间、人数和预算。
- 观演人/证件备注。
- 可接受票档、备选方案、开售时间。
- 可填写大麦项目 ID 或详情页链接，生成官方详情页和移动入口。
- 手机端提供大麦 App 深链、Android intent、官方移动页和网页兜底。

## 合规边界

极速抢索票台只做客户票需整理、官方入口生成和索票单复制。所有订单提交、验证码、实名信息、观演人/乘车人确认和支付都必须由客户本人在官方页面或官方 App 完成。
